# HR Analytics Dashboard

This project is an interactive HR Analytics Dashboard built using Power BI and Excel.

## 📊 Features
- Tracks employee Presence %, WFH %, and SL %
- Time-based trend analysis
- Day-wise performance insights
- Interactive filters and slicers

## 🛠 Tools Used
- Power BI
- DAX
- Power Query
- Microsoft Excel
